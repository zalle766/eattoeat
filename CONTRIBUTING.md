# Contributing to Eat to Eat

Thank you for your interest in contributing to Eat to Eat! We welcome contributions from the community and appreciate your help in making this project better.

### Prerequisites
- PHP 8.2+
- Composer
- Node.js 18+
- MySQL 8.0+
- Git

### Development Setup
1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/eattoeat.git`
3. Install dependencies: `npm run install:all`
4. Set up environment: `npm run setup`
5. Start development servers: `npm run dev`
