# تعليمات تشغيل الموقع

## الطريقة الأولى: استخدام ملف Batch
1. انقر نقراً مزدوجاً على `start_frontend_dev.bat`
2. انتظر حتى يبدأ الخادم
3. افتح المتصفح واذهب إلى `http://localhost:3000`

## الطريقة الثانية: استخدام PowerShell
1. انقر بزر الماوس الأيمن على `start_frontend_dev.ps1`
2. اختر "Run with PowerShell"
3. انتظر حتى يبدأ الخادم
4. افتح المتصفح واذهب إلى `http://localhost:3000`

## الطريقة الثالثة: استخدام Terminal
1. افتح Terminal في مجلد المشروع
2. اكتب: `cd frontend`
3. اكتب: `npm run dev`
4. انتظر حتى يبدأ الخادم
5. افتح المتصفح واذهب إلى `http://localhost:3000`

## الميزات المتاحة:
- ✅ مودال تحديد الموقع عند الدخول لأول مرة
- ✅ بحث المطاعم مع فلترة
- ✅ عرض المطاعم القريبة مع المسافات
- ✅ تصميم متجاوب وجميل
- ✅ بيانات وهمية للمطاعم

## ملاحظات:
- تأكد من أن Node.js مثبت على جهازك
- تأكد من أن npm يعمل بشكل صحيح
- إذا واجهت مشاكل، جرب حذف مجلد `node_modules` وإعادة تثبيت الحزم
